# Cybersecurity Foundations Project:

This project is a beginner-friendly Cybersecurity Foundations application developed in Python. It demonstrates essential cybersecurity concepts such as cryptography, hashing, networking, DNS lookup, port scanning, and password strength analysis.


##  Objectives

* Understand the CIA Triad (Confidentiality, Integrity, Availability)
* Learn cryptographic techniques
* Implement SHA-256 hashing
* Perform encryption and decryption
* Explore networking concepts
* Conduct DNS lookups
* Scan common network ports
* Analyze password strength
* Gain hands-on cybersecurity experience using Python

##  Features

### 1. Cybersecurity Basics

* Introduction to the CIA Triad
* Awareness of common cyber threats:

  * Phishing
  * Malware
  * Ransomware
  * SQL Injection
  * DDoS Attacks
  * Brute Force Attacks

### 2. SHA-256 Hashing

* Generates secure SHA-256 hashes for user-provided messages.
* Demonstrates data integrity verification.

### 3. Encryption & Decryption

* Uses the Fernet module from the Cryptography library.
* Encrypts sensitive information.
* Decrypts encrypted messages securely.

### 4. System Information

* Displays:

  * Operating System
  * OS Release
  * Machine Architecture
  * Processor Information

### 5. Networking Information

* Retrieves:

  * Hostname
  * Local IP Address

### 6. DNS Lookup

* Converts domain names into IP addresses.

### 7. Port Scanner

* Scans commonly used ports:

  * FTP (21)
  * SSH (22)
  * HTTP (80)
  * HTTPS (443)
  * MySQL (3306)
  * And more

### 8. Password Strength Checker

* Evaluates passwords based on:

  * Length
  * Uppercase letters
  * Lowercase letters
  * Numbers
  * Special characters
